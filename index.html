Smart Bird Perch

Designed and built a semi-permanent perch that tracks bird weight in real time. The system is stable on rocky and sandy coastal terrain and supports birds up to 1.6 kg.

Embedded Systems Sensors IoT Conservation

Project Overview

This project involved developing a smart perch for wild coastal birds that can measure weight instantly when a bird lands. The system is designed for rugged outdoor use near the ocean, providing reliable data for conservation research.

Technical Details

  • Load cell sensors for real-time weight measurement
  • STM32 microcontroller for data processing
  • Low-power design for field deployment
  • Stable structure for rocky and sandy terrain
  • Supports birds weighing up to 1.6 kg
