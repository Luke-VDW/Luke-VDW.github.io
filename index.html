My Projects

Line Following Robot

Embedded C project on the STM32 platform. Designed, built, and programmed an autonomous robot capable of detecting and following a line using infrared sensors, PWM motor control, and a PID feedback algorithm.

Embedded C STM32 Robotics PID Control PCB Design

Project Overview

This project involved developing a small-scale autonomous robot to demonstrate real-time control and embedded systems integration. The robot follows a black line on a light surface using an array of infrared sensors to detect deviations from the path. A PID controller adjusts the motor outputs to ensure smooth turns, quick correction, and minimal overshoot. The goal was to create a robust and reliable system that could handle both straight paths and sharp curves.

Technical Details

  • STM32F4 Discovery board as main controller
  • 5x IR reflectance sensors for line detection
  • PID algorithm implemented in Embedded C for real-time error correction
  • Two DC motors with PWM-based speed and direction control
  • Custom PCB for clean sensor integration and motor driver connections

Key Outcomes

  • Improved understanding of control theory and PID tuning in practice
  • Hands-on experience with STM32 peripheral configuration (ADC, timers, PWM)
  • Demonstrated integration of sensors, actuators, and embedded software
  • Applied circuit design skills to create a functional PCB

Rocket Trajectory Controller

Designed and implemented a guidance and control system for a simulated rocket tasked with intercepting a moving asteroid. This project required advanced control theory, nonlinear dynamics modeling, and robust handling of noisy trajectory data.

MATLAB Simulink Control Systems LQR Trajectory Optimization

Project Overview

The objective of this project was to simulate and design a control system capable of guiding a rocket to intercept a moving asteroid. To achieve this, I developed a nonlinear model of the rocket's translational and rotational dynamics and designed a Linear Quadratic Regulator (LQR) to ensure stable, efficient trajectory correction.

Technical Details

  • Nonlinear system modeling of rocket translational and rotational dynamics
  • Linear Quadratic Regulator (LQR) design for optimal trajectory control
  • State-space representation and linearization around equilibrium points
  • Robust handling of noisy asteroid trajectory data

Key Outcomes

  • Successfully intercepted the asteroid within allowed angular constraints
  • LQR controller achieved smooth and efficient corrections with minimal overshoot
  • Demonstrated robustness and reliability under noisy conditions

Smart Bird Perch

Designed and built a semi-permanent perch for coastal bird monitoring that tracks bird weight in real time. The system is rugged enough for rocky and sandy shorelines while remaining sensitive enough to capture subtle weight variations, cormorant birds up to 1.6 kg.

Embedded Systems Sensors IoT Conservation

Project Overview

This project was created in collaboration with a conservation initiative focusing on seabird populations along the South African coast. The perch enables researchers to passively collect accurate weight data whenever birds land, providing insights into seasonal feeding patterns, health trends, and environmental impacts. Its semi-permanent design makes it easy to deploy and relocate without causing environmental damage.

Technical Details

  • ESP32 microcontroller processes sensor input and transmits data
  • Load cell sensors calibrated to measure weights up to 1.6 kg with high precision
  • Designed to withstand harsh, salty, and sandy coastal environments
  • Optimized for extended field deployment with minimal maintenance requirements

Key Outcomes

  • Provides real-time data on bird health without invasive techniques
  • Lightweight and rugged design allows use across various coastal regions
  • Contributes to wildlife monitoring and biodiversity protection
